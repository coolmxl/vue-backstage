# vue-backstage  前后端分离停车管理系统管理员界面

## 后端
```
采用thinkPHP开发，运用mysql数据库，运用第三方包如阿里短信验证
```

### 前端
```
采用Vue框架开发，运用众多依赖。如vue-router、stylus、element-ui、axios等
```

### 功能
```
1.登录、注册
2.管理各个表
  2-1 查看管理员表
  2-2 查看、编辑、删除、修改停车表
  2-3 查看用户表
  2-4 查看、编辑、修改评论表
  2-5 查看头像表
3.添加数据
  3-1 添加用户
  3-1 添加管理员
4. echarts统计网站数据
5. 权限设置
  5-1 登录之后退出清空token权限
  5-2 超级管理员有最高权限


```
